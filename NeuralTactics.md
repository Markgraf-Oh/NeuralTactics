# NeuralTactics Project

## 프로젝트 개요

**프로젝트명:** NeuralTactics

Python으로 전술 전투 게임 시뮬레이션을 구현하고, 이를 기반으로 **전투 AI(Soldier AI)** 와 **전술 AI(Commander AI)** 를 개발하는 기술 검증 프로젝트.  
장기적으로는 언리얼 엔진 등 상용 게임 엔진과 신경망을 연결하는 것을 목표로 하되, 현재는 Python 시뮬레이션 단계에 집중한다.

---

## 프로젝트 목표

1. **계층적 다중 에이전트 AI:** Commander AI → Soldier AI의 지휘 구조로 대규모 전투에서의 전술적 의사결정 구현
2. **모방 학습 → 진화 학습 파이프라인:** 플레이어 데이터 기반 초기 학습 후 유전 알고리즘 + 강화학습으로 심화 학습
3. **두 AI의 상호작용 분석:** 전술 AI 유무에 따른 전투 AI 행동 패턴 비교 연구
4. **게임 엔진 대체 가능한 구조:** Python 시뮬레이션 레이어를 추후 상용 게임 엔진으로 교체 가능하도록 설계

---

## 기술 스택

| 구분 | 기술 |
|------|------|
| 게임 시뮬레이션 | Python (순수 시뮬레이션 로직) |
| 시각화 인터페이스 | Pygame (on/off 가능) |
| 신경망 학습 | PyTorch |
| 핵심 아키텍처 | EGNN + Hybrid (CNN/MLP) |
| 학습 방식 | Behavioral Cloning → Genetic Algorithm + Reinforcement Learning |

## 코드 구조

```
NeuralTactics/
├── common/              # 공용 유틸리티, 데이터 구조, 설정
├── neural/              # 신경망 관련 (모델 정의, 학습, 추론)
│   ├── models/          # EGNN, CNN/MLP 모델 정의
│   ├── training/        # 모방 학습, 유전 알고리즘, 강화학습
│   └── inference/       # 추론 인터페이스
└── game/                # 게임 관련 (추후 게임 엔진으로 대체 가능)
    ├── core/            # 공통 게임 로직 (유닛, 전투 규칙 등)
    ├── simulation/      # 순수 시뮬레이션 (렌더링 없음)
    └── interface/       # Pygame 시각화 인터페이스 (on/off 가능)
```

### 설계 원칙

- **game/core**: 게임 규칙·상태·엔티티 정의. 엔진 종속성 없음
- **game/simulation**: 물리/전투 계산. 순수 Python, 렌더링 없음
- **game/interface**: Pygame 렌더링. 플래그 하나로 비활성화 가능
- **neural**: game 레이어에 의존하지 않음. 입출력은 표준 텐서/배열

## AI 아키텍처

### 계층 구조: Commander - Soldier

```
Commander AI (Macro)
 ├─ 입력: 전체 전장 정보 (아군·적군 위치, 지형, 자원)
 ├─ 출력: 전술 목표 좌표, 우선 타겟, 행동 플래그
 └─ 명령 주입 방식 ↓

Soldier AI (Micro) × N
 ├─ 입력: 로컬 반경 내 유닛 정보 + 지휘관 명령
 └─ 출력: 이동 벡터, 공격 타겟, 스킬 사용 여부
```

### Soldier AI 신경망 구조 (Hybrid Architecture)

#### 이동·회피 모듈 — EGNN (Equivariant GNN)

- 절대좌표 `(x, y)` 를 변환 없이 직접 입력
- 노드 간 상대 벡터·거리를 신경망 내부에서 자동 추출 (동변성 보장)
- 출력: 다음 이동 위치 벡터

```
m_ij  = φ_e(h_i, h_j, ||x_i - x_j||²)        # 거리² 기반 메시지
x_i'  = x_i + Σ_j (x_i - x_j) · φ_x(m_ij)   # 좌표 업데이트 (동변성)
h_i'  = φ_h(h_i, Σ_j m_ij)                   # 피처 업데이트 (불변성)
```

#### 스킬·전술 판단 모듈 — Local Grid + CNN/MLP

- 에이전트 주변 로컬 2D 그리드 생성
- 경량 CNN 또는 MLP로 적 진형 패턴 분석
- 광역기·관통기 등 스킬 선택에 활용

## 학습 파이프라인

### Phase 1 — 모방 학습 (Behavioral Cloning)

| 항목 | 내용 |
|------|------|
| 데이터 | 플레이어 조작 로그 (상태, 이동, 공격 기록) |
| 방식 | 지도 학습 (Supervised Learning) |
| 목표 | 인간 수준의 기본 전술 행동 패턴 학습, 초기 가중치 형성 |

### Phase 2 — 유전 알고리즘 + 강화학습

| 항목 | 내용 |
|------|------|
| 초기 세대 | Phase 1 모방 학습 완료 모델 |
| 개체군 | 동일 모델 가중치에서 파생된 N개 에이전트 |
| 적합도 평가 | 전투 시뮬레이션: 입힌 피해, 생존 시간, 명령 복종도 |
| 선택·교차 | 상위 개체 가중치 교차 |
| 돌연변이 | EGNN 구조 붕괴 방지를 위해 분산을 작게 설정 |

**Fitness Function 구성 요소:**
- 적에게 입힌 총 피해량
- 아군 생존율
- 지휘관 명령 복종도 (목표 지점 도달, 지정 타겟 공격 여부)
- 생존 시간
